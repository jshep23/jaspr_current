# 0.1.0

- Initial port from the [Flutter Current](https://pub.dev/packages/current) package.
- Includes all the core features of the Flutter version (ViewModels, CurrentProperties, CurrentComponents, etc.).

This is an early release. The core patterns and APIs are stable, but there has yet to be any real-world testing of the Jaspr specific components in a fully fledged application project. I have a project on the go that will be using this library extensively, but would love to get some early feedback from the community as well. Until a 1.0 happens, expect some breaking changes, although I will do my best to minimize them.
